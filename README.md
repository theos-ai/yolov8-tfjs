# YOLOv8 with TensorflowJS

<p align="center">
  <img src="./sample.jpg" />

YOLOv8 running locally in your browser using TensorflowJS.

**Run**

```bash
npm install
npm start
```

**Use your own model**

Update `modelURL` in `App.jsx`
   ```jsx
   const modelURL = 'paste-here-your-manifes-url';
   ```
