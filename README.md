YOLOv8 running locally on your browser using tensorflow.js.

**Run**

```bash
npm install
npm start
```

**Use your own model**

Update `modelURL` in `App.jsx`
   ```jsx
   const modelURL = 'paste-here-your-manifes-url';
   ```
